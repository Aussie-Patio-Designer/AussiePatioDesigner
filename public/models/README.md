# Uploaded 3D scene models

Upload these GLB files here so they are served by Next.js at the paths used by the 3D designer:

- `house.glb` → `/models/house.glb`
- `car.glb` → `/models/car.glb`
- `rubbish.glb` → `/models/rubbish.glb`

The designer includes simple fallback geometry, so it will still load before these files are added.
